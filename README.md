# 老魏博客 :link: https://weishuai198.github.io/blog 
💯 Gmeek 一个超轻量级个人博客模板，完全基于Github Pages 、Github Issues 和 Github Actions，可以称作All in Github，
不需要本地部署，从搭建到写作，只需要18秒，2步搭建好博客，第3步就是写作。
## 安装手册：
1【创建仓库】点击通过模板创建仓库，建议仓库名称为XXX.github.io，其中XXX为你的github用户名。

2【启用Pages】在仓库的Settings中Pages->Build and deployment->Source下面选择Github Actions。
3【启用 Pages】在仓库的 Settings 中 Pages->Build and deployment->Source 下面选择 Github Actions。

4【开始写作】打开一篇issue，开始写作，并且必须添加一个标签Label（至少添加一个），再保存issue后会自动创建博客内容，片刻后可通过https://XXX.github.io 访问（可进入Actions页面查看构建进度）。

5【手动全局生成】这个步骤只有在修改config.json文件或者出现奇怪问题的时候，需要执行。

通过Actions->build Gmeek->Run workflow->里面的按钮全局重新生成一次
## 问题处理
1如果有问题可参考[Gmeek快速上手]([https://weishuai198.github.io/blog/](https://blog.meekdai.com/post/Gmeek-kuai-su-shang-shou.html))
2. 在本仓库提交Issues之前，请手动全局生成一次。如果还有错误，提交Issues后，我会帮忙查看构建流程，定位问题出处。
