### 利用Gmeek博客框架在GitHub上创建了我的博客
我想置顶一篇文章，查看官方文档是这样说的
置顶博客文章只需要Pin issue后，手动全局生成一次即可。
可我看几遍也不懂，问AI给出答案
第一步：在 GitHub 仓库中 Pin（置顶）你的 Issue
进入你的 GitHub 博客仓库 XXXXX.github.io
点击 Issues 选项卡。找到你想要置顶的那篇文章
然后在 Issue 页面的右侧菜单栏（或者右上方三个点 ... 的菜单里，取决于 GitHub 的 UI 布局），
找到 Pin issue 选项并点击它。
